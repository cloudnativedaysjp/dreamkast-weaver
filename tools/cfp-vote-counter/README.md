CFP Vote Couter
-----------------------

CFP投票を集計するツールです。

## Requirements

Python3.9+


## Installation

```
python -m venv venv
. ./venv/bin/activate
pip install -r requirements.txt
```

## Usage

Run

```
# Generate CSV
./main.py generate_csv cndt2022 [stg]
```

Help

```
./main.py generate_csv -h
```
